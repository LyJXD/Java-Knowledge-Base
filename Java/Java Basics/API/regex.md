## 正则表达式
### 基本元字符

| **元字符** |       **说明**        |
| :-: | :-------------: |
|  .  |    匹配任意单个字符     |
| \|  |     逻辑或操作符      |
| []  |  匹配字符集合中的一个字符   |
| [^] |     对字符集合求非     |
|  -  | 定义一个区间（例如[A-Z]） |
|  \  |    对下一个字符转义     |
